The Windows 64-bit version of TestDisk & PhotoRec should work under
- Windows XP 64-bit
- Windows 2003 64-bit
- Windows Vista  64-bit
- Windows Server 2008 64-bit
- Windows 7 64-bit

Functionnalities provided by ext2fs and reiserfs libraries are missing:
- TestDisk can't list files from ext2/3/4 and reiser filesystem
- PhotoRec can't recover files from ext2/3/4 free space only, it will recover
files from the whole filesystem.
These functionnalities are available in the Windows 32-bit version.

If you are using a 32-bit version of Windows, get the Windows version of
TestDisk.
If you are using an older version of Windows, run the DOS version of TestDisk.
You can download it from http://www.cgsecurity.org/wiki/TestDisk_Download

TestDisk doesn't need to be installed, you only need to
- extract the files
- run testdisk_win.exe or photorec_win.exe

TestDisk & PhotoRec documentation can be found online:
- http://www.cgsecurity.org/wiki/TestDisk
- http://www.cgsecurity.org/wiki/PhotoRec
